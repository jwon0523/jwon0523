## About

현실 세계를 디지털로 모델링하고 구조를 설계하는 일을 좋아합니다. iOS 앱의 아키텍처와 상태·에러 처리를 직접 설계했고, AI 추천 시스템과 RAG 에이전트로 그 범위를 넓혔습니다. 지금은 같은 역량을 서버에서 발휘하고 있습니다. 클라이언트가 안에서 어떻게 도는지 알고 서버를 설계합니다.

개발할 때 AI를 설계 파트너로 씁니다. 구조와 핵심 로직은 직접 결정하고 구현 속도만 AI로 끌어올립니다.
